# mathextended
Simple lua module aiming to provide more functionalities to the already existing math library of lua.

<table>
  <tr>
    <th>Components</th>
    <th>Definition</th>
  </tr>
  <tr>
    <td><code>math.clamp()</code></td>
    <td>Returns a value limited between a minimum and a maximum.</td>
  </tr>
  <tr>
    <td><code>math.fact()</code></td>
    <td>Returns the factorial of the specified number.</td>
  </tr>
  <tr>
    <td><code>math.sign()</code></td>
    <td>Returns the sign of a number.</td>
  </tr>
  <tr>
    <td><code>math.round()</code></td>
    <td>Rounds a number to the defined amount of decimals. The default decimal amount is 0.</td>
  </tr>
  <tr>
    <td><code>math.lerp()</code></td>
    <td>Interpolates two values.</td>
  </tr>
  <tr>
    <td><code>math.e</code></td>
    <td>The value of Euler's number. Shorthand for math.exp(1).</td>
  </tr>
</table>
